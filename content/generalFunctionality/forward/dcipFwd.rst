Create a DC/IP forward model
============================


To create a DC/IP 3D forward model using the software ``DCIPF3D``. The menu structure is:

- DC: **Create** |rarr| **Forward modelling** |rarr| **DC/IP** |rarr| **3D** |rarr| **DC**

- IP: **Create** |rarr| **Forward modelling** |rarr| **DC/IP** |rarr| **3D** |rarr| **IP**

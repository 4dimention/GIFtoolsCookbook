Run a forward model
===================

Prior to running the forward model, the user must :ref:`write <fwdWriteAll>` the files required to the working directory. Click on the inversion item, select the menu showing its class (e.g., ``MAGforward``):

**[Forward class]** |rarr| **Run [forward name]**

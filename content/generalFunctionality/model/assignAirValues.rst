Assign air values to a model
============================

Select the GIFmodel object then the menu:

**GIFmodel** |rarr| **Edit** |rarr| **Assign value to air cells** 


.. figure:: ../../../images/modelEditMenu.png
    :align: center
    :width: 400

 
Prequisites
-----------

An active cell model associated with the model's mesh.

Delete data columns
===================

Click on desired data object, the menu with its class name (e.g., **MAGdata** for magnetic data) |rarr| **Delete data column(s)**

.. figure:: ../../../images/dataGenEdit.png
    :align: center
    :width: 400


**NOTE:** Data columns that are used in the :ref:`i/o Headers <setioHeaders>` will not show up on the list of columns eligible for deletion.

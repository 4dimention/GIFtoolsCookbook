Create a DC/IP inversion
========================

**2D DC/IP**

To create a DC/IP 2D inversion using the software package ``DCIP2D``. The menu structure is:

- DC: **Create** |rarr| **Inversion** |rarr| **DC/IP** |rarr| **2D** |rarr| **DC**

- IP: **Create** |rarr| **Inversion** |rarr| **DC/IP** |rarr| **2D** |rarr| **IP**

    :align: center
    :width: 400

Create a magnetics inversion
============================

There are three different inversion objects that can be created to invert magnetic data. The first two are using the program ``MAG3D`` with version control (5.0 and 5.1). The last object that can be selected is a magnetic-vector inversion using the program ``MVI``. The menus needed to create these are:

#. MAG3D 5.0: **Create** |rarr| **Inversion** |rarr| **Magnetics** |rarr| **Induced (MAG3D 5.0)**

#. MAG3D 5.1: **Create** |rarr| **Inversion** |rarr| **Magnetics** |rarr| **Induced (MAG3D 5.1)**

#. MVI: **Create** |rarr| **Inversion** |rarr| **Magnetics** |rarr| **MVI (magnetic vector)**

**NOTE**: GIFtools inversion objects will take care of the prerequisite weighting and sensitivity programs prior to inversion for the user.

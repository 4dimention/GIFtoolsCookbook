Set the model unit name
=======================

To set the model unit name, select the GIFmodel object then the menu:

**GIFmodel** |rarr| **Edit** |rarr| **Set unit** 


.. figure:: ../../../images/modelEditMenu.png
    :align: center
    :width: 400


**NOTE** This will *not* change the values of the units (use the :ref:`simple calculator<modelCalculator>` to change units)! Simply the unit name.
